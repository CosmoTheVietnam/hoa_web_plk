# hoa_web_plk

**hoa_web_plk** is a simple static website showcasing a demo project. It provides a basic home page (`index.html`) and can be used as a starting point for further development or as a template for small informational sites.

## 🚀 Demo

You can preview the site locally by opening `index.html` in your browser:

```bash
# from the project root
open index.html        # macOS
# or
xdg-open index.html    # Linux
# or just double-click the file in your file explorer
```

The demo includes a basic layout with placeholder content. Feel free to modify the HTML and add styling or additional pages.

## 📝 Description

This repository is intended for a simple demonstration of a static web page.
- **index.html**: The main entry point for the website.

Currently, the site has minimal structure so it can be extended with CSS, JavaScript, or additional HTML as needed. Use it as a foundation for learning, prototyping, or sharing a lightweight web presence.
